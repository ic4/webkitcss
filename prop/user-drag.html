-webkit-user-drag 属性

设置整个元素而非它的内容可拖动。对应于标签属性draggable。其中图片和链接默认为可拖动（draggable="true"），其他元素默认为不可拖动（draggable="false"）。

1.0+ 3.0+ ? ?

?: 表示版本号未知，但该属性可用。

语法

-webkit-user-drag: auto | element | none

取值

auto 使用默认的拖动行为。图片和链接默认为可拖动，其他元素默认为不可拖动。
element 整个元素而非它的内容可拖动。
none 元素不能被拖动。但可通过复制选中后拖动。
